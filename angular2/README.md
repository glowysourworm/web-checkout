
ag-Grid-ng2-examples - webpack
==============================

Example of using ag-Grid with Angular2 and TypeScript

Building
========

Install Dependencies:

- `npm install`

Dev Build
=========

To build & run:

- `npm start`

Prod Build
=========

To build:

- `npm run build`

e2e Tests
=========

Run Dev Server:

- `npm start`

In a new Terminal, run the tests:

- `npm run e2e`
